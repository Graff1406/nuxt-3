## Use icons

All of icons define in same place `./assets/svg/icons`. For icons exist one file what called sprite.svg (`./assets/svg/sprite.svg`)

- put new icon in directory `./assets/svg/icons`
- run command `npm run svg-sprite-generate`
- use icon component `<ww-icon>name</ww-icon>`. You can check all of accessible icons in section **components -> ww-icon -> gallery**98765t432
