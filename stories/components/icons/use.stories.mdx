# Use icons

---

All of icons define in same place `./assets/svg/icons`. For icons exist one file what called sprite.svg (`./assets/svg/sprite.svg`)

##### Step by step for add a new icon(s)

1. Put new icon in directory `./assets/svg/icons`.
2. Run command `npm run svg-sprite-generate`.
3. Use icon component `<ww-icon>name of icon</ww-icon>`.

### Check All icons

You can view all available icons used in the project in the section [icon gallery](?path=/story/components-icons-gallery--gallery).

If click on a icon card then can copied the icon definition.
